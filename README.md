# Coin IQ - Crypto Prediction and Learning Platform

Coin IQ is a Final Year Project that combines a Next.js web application with a Flask machine learning API for cryptocurrency analytics, price prediction, market insights, news, authentication, user profiles, and learning modules.

## Project Structure

```text
FYPFinal/
├── coin-iq/        # Next.js frontend and backend API routes
├── flask-api/      # Flask ML prediction API
├── Models/         # Trained ML models and prediction exports
└── README.md
```

## Features

- Cryptocurrency dashboard
- Market and coin detail pages
- Prediction dashboard
- Flask-based ML prediction API
- XGBoost, LSTM, LightGBM, and Random Forest model support
- Authentication and user sessions
- Admin panel
- Learning module with courses, quizzes, progress, and certificates
- News and sentiment-related functionality
- Profile management

## Setup On A New Computer

Clone the repository:

```bash
git clone https://github.com/DarknesX22/Final.git
cd Final
```

## Frontend Setup

Go to the Next.js app:

```bash
cd coin-iq
npm install
```

Create a local environment file:

```bash
copy .env.example .env.local
```

If `.env.example` does not exist, create `.env.local` manually and add the required environment variables for database, authentication, OAuth, email, and API URLs.

Run the development server:

```bash
npm run dev
```

The app should run at:

```text
http://localhost:3000
```

## Flask API Setup

Open another terminal and go to the Flask API folder:

```bash
cd flask-api
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

The Flask API will run locally depending on the port configured in `app.py`.

## Environment Variables

The `.env.local` file is not uploaded to GitHub for security reasons.

You must recreate it on every new computer.

Typical values may include:

```env
DATABASE_URL=
JWT_SECRET=
SESSION_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
FACEBOOK_CLIENT_ID=
FACEBOOK_CLIENT_SECRET=
SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASS=
NEXT_PUBLIC_API_URL=
```

Use the same values from your local development setup.
